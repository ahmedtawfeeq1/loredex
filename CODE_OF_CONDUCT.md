# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, no harassment of any kind. Maintainers may
remove, edit, or reject contributions and contributors that violate this.

Report unacceptable behavior privately to the maintainer via the email on the GitHub
profile. Reports are handled confidentially.
